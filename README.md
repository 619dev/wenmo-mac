# 问墨输入法（macOS）

这是问墨 iOS 核心输入逻辑的 macOS `InputMethodKit` 移植版。它完全离线，复用同一份
CC-CEDICT 拼音词库，支持拼音组合、候选选择、空格上屏、退格、Esc 取消及简繁切换。

## 安装

推荐使用 PKG 安装：

1. 双击 `问墨-1.0.6-macOS.pkg`，按系统安装器的提示完成安装。
2. 退出当前 macOS 账户并重新登录，让系统刷新输入法列表。
3. 打开“系统设置”→“键盘”→“文本输入”→“编辑”。
4. 点击“+”，在“简体中文”中选择“问墨”并添加。

安装器会将问墨部署到 macOS 的系统输入法目录 `/Library/Input Methods`，并尝试自动注册、
启用和选择输入源。如果问墨已出现在输入法菜单中，可以跳过第 3、4 步。

## 更新

直接运行新版本 PKG，覆盖安装完成后退出当前账户并重新登录。无需先手动删除旧版本。

## 卸载

1. 在“系统设置”→“键盘”→“文本输入”→“编辑”中移除“问墨”。
2. 在“访达”中选择“前往”→“前往文件夹…”，输入 `/Library/Input Methods`。
3. 删除“问墨.app”（需要管理员权限），然后退出当前账户并重新登录。

问墨是后台输入法组件，不是普通应用程序，请勿将它安装到“应用程序”文件夹或直接双击运行。

## 从源码构建与安装

要求 macOS 13 或更新版本及 Xcode Command Line Tools。

```sh
make
make pkg
make dmg
```

双击生成的 PKG 完成系统级安装。输入拼音后使用空格选择首候选，也可从候选窗选择其他候选。
输入法菜单可切换简繁。

开发时可运行 `make test` 做 Swift 类型检查。应用产物位于 `.build/问墨.app`，安装包位于
`.build/问墨-1.0.6-macOS.pkg`。

## 隐私

输入法不申请网络或录音权限，不包含遥测、广告、远程配置或在线词库。词库只读，组合内容仅
保存在当前输入法进程的内存中。
