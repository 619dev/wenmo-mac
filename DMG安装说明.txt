问墨输入法 1.0.6 安装说明
==========================

注意：问墨是 macOS 输入法，不能安装到“应用程序”文件夹。

1. 在 Finder 菜单中选择“前往”→“前往文件夹…”。
2. 输入 ~/Library/Input Methods 并按回车。
3. 如果 Input Methods 文件夹不存在，请在 ~/Library 下新建它。
4. 将 DMG 中的“问墨.app”拖入 Input Methods 文件夹。
5. 退出当前 macOS 账户并重新登录。
6. 打开“系统设置”→“键盘”→“文本输入”→“编辑”。
7. 点击“+”，在“简体中文”中选择“问墨”并添加。

更新旧版本时，请先在 Input Methods 文件夹中移除旧的“问墨.app”，
再复制新版本，然后退出登录并重新登录。

不要直接双击“问墨.app”；它是后台输入法组件，不是普通应用程序。
